# 黄宾虹数字空间全息集成展演系统 (Digital Humanities Project)

本顶层集成系统将四个关于中国近现代画坛巨匠**黄宾虹**先生的数字人文（Digital Humanities）交互可视化单页子系统进行了深度的系统集成。本项目通过全景式、多维度的时空叙事交互，旨在解决传统单一维度数字档案割裂、可视化系统各自分散的问题，为学者及大众提供了一站式、浸润式的黄宾虹数字人生与艺术研究空间。

---

## 🧭 核心集成模块

系统通过顶部的“新国风”水墨微调标签导航栏（Tabs）控制，**0ms延时无缝切换**以下四个独创的数字人文子系统：

| 模块名称 | 对应文件名 | 核心数字人文技术与功能简述 |
| :--- | :--- | :--- |
| **画作交互展览** | `1.html` | 数字化交互展厅。采用卡片流体布局与空间错落倾斜的端庄国风美学视觉，配合自由聚焦、3.2倍无损悬停放大交互，直观呈现黄宾虹各时期代表画作与水墨墨法演变。 |
| **时空交游人脉图** | `黄宾虹时空交游人脉地理网络图-v4(1).html` | 基于 **D3.js** 驱动的社会网络与人脉空间关联图谱。将黄宾虹一生中交往的近百位文化名流（齐白石、张大千等）抽象为网络节点，并动态映射到其一生的空间漂移地理坐标中。 |
| **生命艺术之河流** | `时间画卷.html` | 基于 **ECharts** 深度定制的全息数字生命时间长河。将黄宾虹九十载的生命历程化作横向流淌的艺术河流，通过大数据轨迹实现历史事件、创作时段的横向对齐与纵向微观下钻。 |
| **时空多维画卷** | `数字人生：黄宾虹.html` | **3D 翻页交互空间舞台**。通过多维透视卷轴、SVG 节点连线交互，将宏观的历史年谱与微观的“画论观点”、“交游轶事”通过浮层弹窗进行全息联动，打造沉浸式阅读空间。 |

## 🚀 性能与渲染架构设计

### 1. 为什么它如此顺畅？
由于各个子系统携带了成千上万行 D3 节点数据、ECharts 复杂的轨迹坐标以及数万字历史档案，如果直接将其压缩为单文件 Base64 会导致浏览器频繁进行**巨量数据实时解码**，造成严重的假死和切换卡顿。

### 2. 核心优化机制：
* **DOM 常驻与解耦**：主框架（`index.html`）使用标准 `<iframe>` 标签直接链接 4 个本地子系统。所有页面在主框架初次打开时，即由浏览器在后台线程进行**静默并发预加载**并完成复杂的 JS 逻辑（如 D3 引力布局计算）。
* **物理隐藏（Visibility 机制）**：点击顶部的切换按钮时，底层代码不使用会触发页面重绘（Re-render）的解构刷新，而是通过 CSS 的 `visibility` 与 `opacity` 属性进行纯粹的硬件加速过渡显示。
* **状态持久化**：切换过程**不丢失任何子系统的当前交互状态**（例如：在“人脉图”中点选了某个画家，切换去“河流”再切回来，之前的点选和放大状态依然完美保留）。

## 📂 项目目录结构要求

```text
📁 黄宾虹数字人文项目根目录/

├── 1.html                               <-- 1. 画作交互展览

├── 时间画卷.html                         <-- 3. 生命艺术之河流
