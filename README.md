# 数字人文：数字黄宾虹

本项目包含了 4 个相互独立的纯前端网页文件（HTML/CSS/JS）。它们分别通过不同的图表库和交互方式，展示了画家**黄宾虹**先生的画作、人脉地理轨迹、生平时间轴以及多维年谱。

项目下载到本地后，直接双击任意 `.html` 文件即可在浏览器中运行。

---

## 📂 页面功能与技术实现说明

### 1. 画作交互展览厅 (`黄宾虹作品展示.html`)
* **实际功能**：画作瀑布流展示与局部细节放大。
* **技术看点**：页面采用了卡片流体布局，每个画作卡片在初始化时会产生微小的随机倾斜角度。当鼠标悬停（`mouseenter`）在某幅画作上时，该卡片会触发 `active` 样式，在原位实现 **3.2 倍的无损稳定放大**，便于清晰观察黄宾虹笔墨的皴法与细节。

### 2. 时空交游人脉地理网络图 (`黄宾虹时空交游人脉地理网络图.html`)
* **实际功能**：展示黄宾虹与同时代文化名人（齐白石、张大千等）的社交网络，并联动地图展示他们的地理轨迹。
* **技术看点**：基于 **D3.js (v7.8.5)** 开发。页面将人物抽象为拓扑图的节点（Nodes），利用 D3 的引力仿真（`d3.forceSimulation`）计算节点排斥与拉伸。点击某个人物时，会动态生成 SVG 二次贝塞尔曲线（`Q` 路径命令），将人物连线映射到对应的地理坐标标记上。

### 3. 黄宾虹生平事迹多维可视年谱 (`全景图.html`)
* **实际功能**：基于时间轴的高维综合年谱图表，并支持局部点击下钻加载外部展览页。
* **技术看点**：外层容器采用模块化自适应布局（`max-width: 95vw`），核心内容是由复杂的数据序列组成的横向扩展年谱。背景选用中式复古色（`#fdfbf7`），结合高对比度的深色边框衬托。

### 4. 时空多维画卷 (`数字人生：黄宾虹.html`)
* **实际功能**：中式卷轴风格的 3D 翻页阅读空间与年谱联动。
* **技术看点**：纯 CSS3 3D 舞台构建。利用 `perspective: 2500px` 视距与 `transform-style: preserve-3d` 属性，在网页上模拟出实体书卷翻页的视觉深度。内置 SVG 节点连线交互，点击年谱中的“画论”、“轶事”等高亮文本，可通过原生 JS 直接唤醒对应的弹出式详情遮罩层（Overlay）。
