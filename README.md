# 🏛️ 黄宾虹全息数字人文空间图谱集群 (Digital Humanities Project)

本项目是以中国近现代画坛巨匠、美术理论家**黄宾虹**先生（1865－1955）为核心研究对象的数字人文（Digital Humanities）交互可视化基础设施集群。

项目打破了传统单一维度的数字文本档案割裂，通过 **数字化交互展厅、D3.js 动态关联图谱、ECharts 时间序列长河、3D 空间多维视卷** 4 个独立系统，从宏观到微观、从时空到位移，全景式构建了一个浸润式的学术研究和公众展演空间。

---

## 🗺️ 数字人文系统矩阵（项目集群清单）

本仓库存放的 4 个 `.html` 文件为各自独立的、高度定制的可视化大屏系统，分别从不同的学术视角切入黄宾虹的艺术生命：

### 1. 🎨 画作交互展览厅 (`1.html`)
* **学术视角**：微观艺术感知与墨法演变。
* **功能简述**：数字化交互展厅。采用卡片流体布局与空间错落倾斜的端庄国风美学视觉，配合自由聚焦、3.2倍无损悬停放大交互，直观呈现黄宾虹各时期代表画作（如“黑宾虹”、“白宾虹”时期）的笔墨细节与神韵。

### 2. 🌐 时空交游人脉地理网络图 (`黄宾虹时空交游人脉地理网络图-v4(1).html`)
* **学术视角**：宏观社会网络（SNA）与地理空间位移（Spatial Turn）的交织。
* **功能简述**：基于 **D3.js** 强力驱动的社会网络与人脉空间关联图谱。将黄宾虹一生中与之交往的近百位文化艺术名流（如齐白石、张大千、柳亚子等）抽象为网络节点，并动态映射到其一生的空间漂移地理坐标中，揭示了二十世纪上半叶中国精英文化圈的迁移图景。

### 3. 🌊 生命艺术之河流 (`时间画卷.html`)
* **学术视角**：线性时间序列与微观历史事件的无缝下钻。
* **功能简述**：基于 **ECharts** 深度定制的全息数字生命时间长河。通过大数据轨迹与 Papaparse 驱动，将黄宾虹九十载的生命历程化作横向流淌的艺术河流，实现历史事件、政治背景、创作时段的横向对齐，支持多级缩放与视口同步。

### 4. 📜 时空多维画卷 (`数字人生：黄宾虹.html`)
* **学术视角**：多维叙事空间与三维多卷轴交互。
* **功能简述**：**3D 翻页交互空间舞台**。通过多维透视卷轴、SVG 节点连线交互，将宏观的历史年谱与微观的“画论观点”、“交游轶事”通过浮层弹窗进行全息联动，提供极具仪式感的沉浸式阅读空间。
